# DigitechLifestyle

**The Future of Digital Living, Wealth, AI & Automation.**

DigitechLifestyle is being built as a modern content, affiliate, newsletter, and digital-product platform focused on AI tools, automation, digital wealth, smart lifestyle technology, and online business systems.

## Business Model

- SEO blog articles
- Affiliate marketing
- AI tool directory
- Digital downloads and templates
- Email newsletter
- Sponsored content
- Automation consulting offers
- Google AdSense-ready ad zones

## Core Site Sections

1. AI Tools
2. Digital Wealth
3. Automation
4. Smart Lifestyle
5. Make Money Online
6. Tool Directory
7. Blog
8. Newsletter
9. Resources

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- SEO metadata
- Content-ready data structure
- Mobile-first responsive layout

## Local Setup

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Deployment

Recommended deployment options:

- Vercel for fastest Next.js deployment
- Hostinger VPS or web app hosting for custom hosting
- WordPress/Hostinger can be used later if converting this into a WordPress theme

## Monetisation Setup

Replace placeholder affiliate links, newsletter form action, and AdSense blocks before launch.
