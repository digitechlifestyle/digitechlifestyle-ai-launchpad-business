# AI Launchpad for Business

AI Launchpad for Business is a SaaS-style platform concept that helps companies understand where AI can save time, reduce costs, and improve operations.

## What it does

- Scans business information and website details
- Generates an AI readiness score
- Estimates potential savings
- Recommends AI opportunities
- Shows a 30 / 60 / 90 day AI implementation roadmap
- Reassures customers with privacy and data-security messaging

## Tech stack

- Next.js
- React
- Inline styling for the first MVP preview

## Run locally

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```
